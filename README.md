# TécnicoYa - Plataforma de Servicios del Hogar

Bienvenido a TécnicoYa, la primera plataforma dominicana al estilo inDrive para conectar clientes con técnicos certificados.

## 📂 Archivos

- **tecnicoYa.html** - Página principal (landing page)
- **solicitar-servicio.html** - Formulario para clientes que necesitan servicios
- **registro-tecnico.html** - Formulario de registro para técnicos

## 🚀 Características

✨ Diseño estilo Apple - minimalista y elegante
🎨 Animaciones suaves y micro-interacciones
📱 Totalmente responsive (móvil, tablet, desktop)
💬 Integración con WhatsApp
⚡ Rendimiento optimizado

## ⚙️ Configuración en GitHub Pages

1. Crea un repositorio en GitHub llamado `tecnicoya`
2. Sube estos archivos
3. Ve a Settings → Pages
4. Selecciona "main" como rama
5. Tu sitio estará en: `https://tuusuario.github.io/tecnicoya`
